# Rodrigo Raw

**`Desenvolvedor FullStack`**

Meu nome é Rodrigo Vieira Santos Raw, sou um estudante de Análise e Desenvolvimento de Sistemas (ADS) na Universidade Santa Cecília, apaixonado por construir aplicações web e jogos.
